## 📌 About The Project
StudySync is a fully responsive, modern landing page designed for an educational technology platform. It features a clean user interface, smooth animations, and a mobile-first approach. This project was built to showcase modern web design principles without relying on heavy frameworks.

## ✨ Key Features
* **Fully Responsive Design:** Adapts seamlessly to all screen sizes (Mobile, Tablet, Desktop).
* **Interactive Navigation:** Custom-built hamburger menu for mobile devices using Vanilla JavaScript.
* **Modern CSS Layouts:** Extensive use of CSS Flexbox and Grid for structured, clean sections.
* **Smooth UI/UX:** Hover states, active button press effects, and smooth layout transitions.
* **Clean Architecture:** Use of CSS variables (`:root`) for easy theme management and maintainability.

## 🛠️ Built With
* **HTML5** (Semantic structure)
* **CSS3** (Custom properties, animations, Flexbox, Grid)
